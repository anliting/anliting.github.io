在萬維網中的動畫

作者：丁安立

在萬維網中，我們可以利用這三個東西來製作動畫：

  • document.timeline.currentTime
  • requestAnimationFrame
  • cancelAnimationFrame

範例 0：碼表

製作一個碼表，它有「開始」與「停止」兩個按鈕。在每次「開始」按鈕被按下之後，在下一次的「停止」按鈕被按下之前，它會實時地顯示從上次按下「開始」按鈕到現在過了多少毫秒。

像這個樣子： 

let afr=0
let span=document.createElement('span')
let startButton=document.createElement('button')
startButton.onclick=()=>{
    if(afr)
        return
    startButton.disabled=true
    stopButton.disabled=false
    let z=document.timeline.currentTime
    let animate=t=>{
        span.textContent=''+Math.floor(t-z)
        afr=requestAnimationFrame(animate)
    }
    afr=requestAnimationFrame(animate)
}
startButton.textContent='開始'
let stopButton=document.createElement('button')
stopButton.disabled=true
stopButton.onclick=()=>{
    if(!afr)
        return
    startButton.disabled=false
    stopButton.disabled=true
    cancelAnimationFrame(afr)
    afr=0
}
stopButton.textContent='中止'

其實大多數的動畫都與碼表的原理類似。一個砲彈飛行的動畫，繪製了時間的二次函數的碼表；一部電影，只是一個每一個瞬間都是一幅美麗畫像的碼表。

其他 0：不動點組合子

這個東西我也不是很懂；不過我覺得這麼寫挺有趣的。

「範例 0：碼表」中的 

let animate=t=>{
    console.log(t-z)
    afr=requestAnimationFrame(animate)
}
afr=requestAnimationFrame(animate)

可以利用不動點組合子寫成這個樣子： 

afr=requestAnimationFrame(
    (a=>(b=>a(a=>b(b)(a)))(b=>a(a=>b(b)(a))))
    (animate=>t=>{
        console.log(t-z)
        afr=requestAnimationFrame(animate)
    })
)
